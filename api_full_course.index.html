HTTP Status Code Reference

200OK — Request succeeded
201Created — Resource created successfully
204No Content — Deleted, nothing to return
400Bad Request — Malformed request / invalid data
401Unauthorized — Missing or bad credentials
403Forbidden — Authenticated but not authorized
404Not Found — Resource doesn't exist
429Too Many Requests — Rate limit hit
500Internal Server Error — Server-side bug
